定制并下载 Bootstrap

通过自定义 Bootstrap 组件、Less 变量和 jQuery 插件，定制一份属于你自己的 Bootstrap 版本吧。此功能支持 IE9+ 或最新版本的 Safari、Chrome 或 Firefox 浏览器。

有现成的配置文件？上传并导入你的 config.json 文件吧。

没有现成的配置文件？没关系，从下面开始自定义各字段吧。

Less 文件

选择将哪些 Less 文件编译到你自己的定制 Bootstrap 版本中。不确定每个文件的用途吗？请先通读此文档中的 全局 CSS 样式 和 组件 章节。

Magic

生成的两个文件

所有选中的插件都将会编译为一份是和阅读的版本 bootstrap.js 和一份压缩版本 bootstrap.min.js。我们建议将压缩版本用于生产环境。

依赖 jQuery

所有插件都依赖最新版本的 jQuery，因此，必须首先将 jQuery 引入页面。

Less 变量

通过 Less 变量可以自定义颜色、尺寸等，最终将会反映到你所下载的 CSS 样式表文件中。
