## 总结

### Boolean 属性

| 属性        | 描述                                     |
| ----------- | ---------------------------------------- |
| constructor | 返回创建 JavaScript Boolean 原型的函数。 |
| prototype   | 允许您向 Boolean 原型添加属性和方法。    |

### Boolean 方法

| 方法       | 描述                               |
| ---------- | ---------------------------------- |
| toString() | 将布尔值转换为字符串，并返回结果。 |
| valueOf()  | 返回布尔值的原始值。               |
